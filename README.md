# transportation

A Flutter project about transportation system among Egypt

## In this project
- mvvm pattern
- drawer
- dropDown menu
- app router
- listView
- gridView
- image gallery saver
- dio
- state management
- flutter bloc
- dartz
- flutter launcher icons
- shared preferences
- animated splash screen
